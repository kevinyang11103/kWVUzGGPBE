# 社团管理系统 java883

## 项目概述

社团管理系统是一款基于前后端分离架构的软件，旨在提高社团管理的效率和便捷性。系统采用Springboot和Vue技术栈，集成了源码、数据库脚本及详尽的报告文档。

## 技术栈

- **前端技术**：Vue、Element-UI
- **后端技术**：Springboot、MyBatisPlus
- **数据库**：MySQL
- **开发语言**：Java

## 功能模块

### 管理员后台

- **用户管理**：对系统用户进行管理。
- **社团管理**：对各个社团的基本信息进行管理。
- **社团类型管理**：管理社团的分类信息。
- **社团成员管理**：对社团成员信息进行管理。
- **通知信息管理**：发布与管理通知信息。
- **入团申请**：管理成员的入团申请。
- **费用管理**：对社团相关费用进行管理。

### 社团成员后台

- **社团信息管理**：查看和管理社团的基础信息。
- **社团成员管理**：社团内部成员的信息管理。
- **入团申请**：成员可发起或查看入团申请状态。
- **社团活动管理**：发布与跟踪社团的活动。
- **通知信息管理**：接收和查看社团通知。

## 系统角色

系统定义了两个角色：管理员和社团成员，分别拥有不同的权限和功能。

## 运行环境

- **服务器**：推荐使用Java推荐配置的服务器环境。
- **数据库**：MySQL 5.7 或更高版本。
- **浏览器**：支持现代Web标准的主流浏览器。

## 部署步骤

1. 安装并配置好所需的服务器环境和数据库。
2. 导入数据库脚本。
3. 部署后端服务，确保Springboot应用可以正常运行。
4. 部署前端应用，确保与后端服务能够正常通信。
